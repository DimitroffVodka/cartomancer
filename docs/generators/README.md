# Watabou Map Generators — Community Usage Guides

This is an **unofficial, community-written** set of usage guides for **Oleg Dolya's (Watabou)** free browser-based map generators. They exist to fill a real gap: the generators are wonderfully deep, but there is no good reference explaining what all the **right-click menu** settings actually do. These pages walk through every menu, dialog, and toggle so you can drive the tools confidently instead of guessing.

## About this documentation

This wiki accompanies the **Cartomancer** Foundry VTT module, which imports maps from these generators directly into your scenes. But the guides are **tool-first** — they describe the generators on their own terms, so they are equally useful to anyone using the generators directly in a browser, regardless of whether you use Foundry or Cartomancer at all.
